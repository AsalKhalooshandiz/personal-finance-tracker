# Personal Finance Tracker

Ich habe dieses kleine Python-Projekt gemacht, um meine eigenen Finanzen besser zu organisieren.  
Mit dem Programm kann man Einnahmen und Ausgaben speichern.  
Man sieht alle Transaktionen auf einen Blick und bekommt den aktuellen Kontostand angezeigt.  
Am Ende werden die Daten in einer JSON-Datei gespeichert, damit nichts verloren geht.

---

### Was kann das Programm?

- Einnahme hinzufügen  
- Ausgabe hinzufügen (als negativer Betrag)  
- Alle Transaktionen tabellarisch anzeigen  
- Nach Kategorien filtern  
- Gesamtsaldo berechnen  
- Daten automatisch laden und wieder speichern  

---

### Wie funktioniert das technisch?

Ich habe dafür einfache Python-Grundlagen benutzt:

- Listen und Dictionaries für die Daten
- Schleifen, um durch alle Einträge zu gehen
- if-Bedingungen für Entscheidungen
- Funktionen, um den Code übersichtlich zu halten
- Fehlerbehandlung mit `try/except`
- Dateizugriff mit JSON
